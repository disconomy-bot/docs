Economy Setup

Initial Setup

Run /guild setup to set up your guild so that members can set your guild as their primary guild and work for your guild. Be prepared to create a unique ticker for your guild (a 4-6 character identifier) to identify and represent your guild.

Work Setup

After running the initial /guild setup, members can work for your guild to allow the guild to use labor hours to produce factory goods.

Commands

  • /guild wage set-default: Set the default member wage.
  • /guild wage set: Set the wage for an individual user.
  • /guild wage set-role: If your guild is premium, set role-based wages.

Tips

  • Ensure fair wages to motivate members to work effectively for the guild.
  • Monitor and adjust wages periodically based on guild performance and needs.

Item Production

Using the raw materials purchased or contracted from members, along with labor hours produced by members from working, you can produce manufactured goods via /guild produce.

Commands

  • /item: View the materials and requirements for producing a specific good.
  • /contract create: Create a contract to have members retrieve items for your guild.

Tips

  • Plan production carefully to maximize efficiency and minimize waste of resources.
  • Use contracts to streamline the collection of necessary raw materials.

Guild Shop

You can create custom items and add them to your guild shop. You can also allow members to purchase roles through the guild shop. To do either, run /guild shop create-item.

Tips

  • Design items that are valuable and appealing to your members.
  • Regularly update the shop to keep it engaging for your guild.

Configuring Work Features

Setting up work-related features involves enabling members to earn and manage currency. Hereâ€™s how to get started:

Commands

  • /guild chatmoney set-rate <rate>: Set the earning rate for members based on activity.
  • /guild chatmoney view-rate: View the current earning rate.
  • /guild chatmoney add-whitelist <channel>: Specify channels where members can earn currency.
  • /guild chatmoney add-blacklist <channel>: Exclude specific channels from earning currency.

Example

To set the earning rate to 5 coins per message:

/guild chatmoney set-rate 5

Tips

  • Use blacklists and whitelists to control where currency can be earned, ensuring fairness and minimizing spam abuse.
  • Monitor activity periodically to adjust rates as needed.

Audit Logs

Keep track of guild activities with audit logs. Configure the logs to record important events for transparency and management.

Commands

  • /guild auditlog set-channel: Set the channel where audit logs will be posted.
  • /guild auditlog set-events: Configure which events to log. Options include: all, transfer, shop_purchase, currency_exchange, and global_sale.
  • /guild auditlog view-channel: View the current channel for audit logs.
  • /guild auditlog view-events: View the currently configured events to log.

Tips

  • Regularly review audit logs to ensure proper use of guild resources and identify any suspicious activity.
  • Limit log access to trusted members to maintain confidentiality.

Currency Exchange

Facilitate the conversion of local guild currency to global economy currency and vice versa.

  • /guild currency-exchange exchange: Exchange your server currency to Global Disconomy currency or vice versa.
    • Usage: Specify the amount and direction of exchange. Example: /guild currency-exchange exchange [IN -> OUT] || [OUT -> IN].
  • /guild currency-exchange set-exchange-rate: Set the guild currency exchange rate (global to local and local to global).
    • Usage: Define both rates explicitly. Example: /guild currency-exchange set-exchange-rate [IN => OUT] || [OUT => IN].
  • /guild currency-exchange view: View the guild currency exchange rates.
    • Usage: No parameters needed; simply run the command to view rates.
  • /guild fund amount: Donate global currency to the guild bank to be used by the guild.
    • Usage: Specify the donation amount. Example: /guild fund 500.
