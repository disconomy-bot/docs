Disconomy Bot Documentation - Base Setup

Introduction

Welcome to the setup guide for Disconomy. This guide will walk you through setting up role permissions and preparing your server for effective use of the bot's work-related features.

Setting Up Role Permissions

Role permissions in Disconomy allow you to designate specific roles to manage guild economy features. Follow these steps to configure role permissions:

Commands

  • /guild rolepermissions set <role>: Assign a role with permissions to manage the guild economy.
  • /guild rolepermissions view: View all roles currently assigned with permissions.
  • /guild rolepermissions remove <role>: Revoke permissions from a specific role.

Example:

To give the "Manager" role permission, use:

/guild rolepermissions set Economy Manager <manage_all: boolean> <manage_funds: boolean> <manage_workers: boolean>

Tips

  • Ensure only trusted roles have these permissions to avoid unauthorized changes to the economy setup.
  • Regularly review the permissions using /guild rolepermissions view.

Work Setup

After running the initial /guild setup, members can work for your guild to allow the guild to use labor hours to produce factory goods. Go here to learn how to setup your guild: guild setup

Commands

  • /guild wage set-default: Set the default member wage.
  • /guild wage set: Set the wage for an individual user.
  • /guild wage set-role: If your guild is premium, set role-based wages.

Tips

  • Ensure fair wages to motivate members to work effectively for the guild.
  • Monitor and adjust wages periodically based on guild performance and needs.

Gambling & Casino Setup

Guild members can engage in various gambling activities, which now use local currency and can be configured by guild admins.

/guild config set-casino-config: Configure betting limits and enable or disable specific gambling commands.
    Parameters:
  • bet_limit: Set a maximum bet limit for all gambling activities.
  • all: Enable or disable all gambling games at once.
  • blackjack: Toggle whether blackjack is available for members.
  • roulette: Toggle roulette availability in the guild.
  • poker: Enable or disable poker (once released).
  • slots: Enable or disable slot machines (once released).

Usage Example: /guild config set-casino-config bet_limit:1000 blackjack:True roulette:False poker:True slots:False

  • Tips: Adjust limits based on your guild’s economy to maintain balance and prevent excessive betting.
  • Disable specific games if you want to limit gambling activities within your guild.

Guild Config

Guild admins can configure various aspects of their guild economy using the following commands:

  • /guild config set-currency-symbol <currency>: Set the local economy currency symbol. It can also be emojis!
    • Usage: Define the currency symbol that will represent the guild’s economy. Example: /guild config set-currency-symbol $.
  • /guild config set-start-balance <balance>: Set the balance that members start with.
    • Usage: Specify the starting balance for new members. Example: /guild config set-start-balance 1000.
  • /guild config show-leaderboard-position <show>: Configure whether leaderboard positions are shown in balances.
    • Usage: Toggle leaderboard position visibility. Example: /guild config show-leaderboard-position True.
