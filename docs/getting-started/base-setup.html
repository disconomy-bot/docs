Disconomy Bot Documentation - Base Setup

Introduction

Welcome to the setup guide for Disconomy. This guide will walk you through setting up role permissions and preparing your server for effective use of the bot's work-related features.

Setting Up Role Permissions

Role permissions in Disconomy allow you to designate specific roles to manage guild economy features. Follow these steps to configure role permissions:

Commands

  • /guild rolepermissions set <role>: Assign a role with permissions to manage the guild economy.
  • /guild rolepermissions view: View all roles currently assigned with permissions.
  • /guild rolepermissions remove <role>: Revoke permissions from a specific role.

Example:

To give the "Manager" role permission, use:

/guild rolepermissions set Economy Manager <manage_all: boolean> <manage_funds: boolean> <manage_workers: boolean>

Tips

  • Ensure only trusted roles have these permissions to avoid unauthorized changes to the economy setup.
  • Regularly review the permissions using /guild rolepermissions view.

Work Setup

After running the initial /guild setup, members can work for your guild to allow the guild to use labor hours to produce factory goods. Go here to learn how to setup your guild: guild setup

Commands

  • /guild wage set-default: Set the default member wage.
  • /guild wage set: Set the wage for an individual user.
  • /guild wage set-role: If your guild is premium, set role-based wages.

Tips

  • Ensure fair wages to motivate members to work effectively for the guild.
  • Monitor and adjust wages periodically based on guild performance and needs.
